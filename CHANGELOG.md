# Changelog

All notable changes to the Zed One theme will be documented in this file.

## [Unreleased]

### Added
- **Zed One Light** - Major new light theme variant
- **Zed One Dark** - Refined dark theme iteration
- Uses Zed's v0.2.0 schema
- More comprehensive syntax highlighting
- Version control colors updated for split diff/word diff
- Improved diagnostic colors

### Changed
- Migrated from `zed-dark.json` to `zed-one.json` as the primary theme file
- Updated theme naming from "Zed Dark" to "Zed One" with both Dark and Light variants

### Removed
- Legacy `zed-dark.json` file (replaced by `zed-one.json`)

### Notes
- Theme is currently a work in progress
- Some edge cases may not be fully styled yet
- Subject to change at any time
